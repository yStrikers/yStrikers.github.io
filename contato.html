  • Método Spoofing

    Presos os suspeitos de invadir diversos celulares de autoridades de forma ilícita,contaram a Polícia Federal qual o método foi usado para fazer a simples invasão aos celulares.Baseia-se na técnica do Caller ID Spoofing, o método utilizado pelo grupo. ... Caller ID Spoofing é uma técnica que faz um celular, ou mesmo um computador ligado à rede de telefonia, fingir ser o smartphone com aquele determinado número.O hacker fazia diversas chamadas para o telefone deixando assim a linha congestionada, neste momento solicitava uma chamada de aúdio do telegram para logar na conta.Com a linha ocupada a chamda com o código ia parar na caixa postal.Posteriormente ele ligava para a caixa postal da vítima e ouvia o código para login.
